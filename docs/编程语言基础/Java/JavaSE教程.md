## 课程简介

****

- 先修要求：零基础
- 课程作者：柏码程序员-首席讲师
- 课程时长：29h
- 内容简介：针对于大学生录制的Java学习课程（含JavaSE和数据结构与算法）

**课程评价**：

* 该课程细致教学了Java8语法基础和数据结构的内容
* 相较其他课程，该课程不仅使用了更现代化的Idea编译器，且细致剖析了集合类及高级特性
* 推荐学习这门课程前至少学过一门面向对象的语言或了解相关概念
* JavaSE的内容是Java技术栈开发的基础，建议细致学习本课程并进行大量练习
