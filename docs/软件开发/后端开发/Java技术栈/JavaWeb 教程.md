* 这门课程包含了计算机网络基础、数据库、依赖管理、前端基础、JSP 开发等技术。
* 相比于某马某硅谷等课程，该课程从零基础开始教学，不超前且细致教学了 JSP 开发流程。
* 学习这门课程时，大量的实战内容巩固使得在我学完后可以独立进行一些简单 web 应用的开发。
* 我惊讶于该视频作者的用心良苦，能把繁琐和复杂的技术讲解得通俗易懂。
